Raigad Mix Boxing Association Website

Files included:
- index.html
- styles.css
- script.js
- assets/logo.jpg (your uploaded logo used in navbar, hero, and footer)
- local SVG placeholder graphics for gallery, founder, hero poster, and champion cards

How to run:
1. Open index.html in any modern browser.
2. For best autoplay compatibility with the hero background video, you can later replace the YouTube iframe with a local MP4 if needed.

Notes:
- Home page contains About, Associated Links, and Gallery sections.
- Separate tabs/pages exist for Organization, Founder, Champion, Calendar, Contact Us, and Community.
- The contact form is demo-only and needs backend/email integration for live submissions.
